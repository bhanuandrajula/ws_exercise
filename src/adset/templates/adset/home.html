    Geo Locations: This menu consists of two drop downs

    1. Geo location by
    2. Top frequent locations

    PS: In case of Geo location by Zips, the corresponding region id is mapped and displayed

    Data:This Menu is used to load data from the source This uses a local json which will be updated based on delete operations, Data can always be overwritten from source using Load This also provides feature to delete the target using target id Deletion of subtypes can be done using Delete by subtype After deleting target or subtypes this can be cross verified on clicking view button and searching for deleted value
